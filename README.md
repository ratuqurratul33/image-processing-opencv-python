# Praktikum Sistem Multimedia

Repository ini berisi hasil praktikum pengolahan citra menggunakan Python dan OpenCV.

## 📂 Isi:
- Diskusi.ipynb → Jawaban pertanyaan diskusi
- TugasMandiri.ipynb → Implementasi kode dan hasil pengolahan citra
- pemandanganSample.jpg → Gambar yang digunakan

## 🛠️ Tools:
- Python
- OpenCV
- Matplotlib

## ⚙️ Fitur:
- Menampilkan gambar
- Konversi ke grayscale
- Resize dan crop gambar
- Histogram grayscale
